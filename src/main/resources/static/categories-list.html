最佳周末烤土豆

  • by Admin
  • 3 min read
  • 20 Comment

土豆洗净，擦干，表皮抹油，入烤箱200度45分钟，土豆剖开，两半或四半，将瓤挖出，注意，多挖些，薄些好吃，依次在挖好的坑里填入火腿，入烤箱再10分钟

蔬菜沙拉最佳做法

  • by Admin
  • 3 min read
  • 20 Comment

先把薄荷叶切碎，与优格、盐、胡椒一同拌匀。接着将清洗干净的生菜叶排放在盘底，并放少许玉米粒，然后再加入小黄瓜、圣女蕃茄、三椒丝。把鸡肉丁炒熟拌入，淋上薄荷优格酱汁即可。

自制布丁

  • by Admin
  • 3 min read
  • 20 Comment

1、将QQ糖放到碗里，开水倒至一半，放到微波炉里加热1分钟。 2、将牛奶倒入放QQ糖的碗里，边放边搅拌。 3、直接放入冰箱冷藏。如果想弄成各种形状的，可以用汤勺把牛奶和QQ糖的混合物盛到模具里再冷藏。

早餐

自制三明治

  • by Admin
  • 3 min read
  • 20 Comment

火腿肠切丁。黄瓜切丁（我也只切了三分之一的黄瓜）。白煮蛋切丁。把火腿丁、黄瓜丁、白煮蛋丁全部倒入大碗中。放入沙拉酱，拌匀。拿出一片的白土司，放上拌好的沙拉。再放一片白土司，放上拌好的沙拉。再后再在上面放一片土司，就做好了，再用刀在对角线处切开，成两个三角形。

牛奶布丁

  • by Admin
  • 3 min read
  • 20 Comment

将鸡蛋倒入碗中搅散。加入牛奶搅拌均匀，调入白砂糖。放入容器中，盖上保鲜膜，上锅蒸20分钟左右即可。

饮料

酸甜可口的酸奶

  • by Admin
  • 3 min read
  • 20 Comment

将纯牛奶倒入锅中加热。加热的过程中，往牛奶中加入酸奶和适量的糖，搅拌均匀。将牛奶盛出，放入密封的容器内即可。然后放在太阳不能直射的地方，进行发酵。时间到了即可加入自己喜欢的水果一起吃。

热可可

  • by Admin
  • 3 min read
  • 20 Comment

牛奶放打蛋盆里微波炉加热中高火半分钟，巧克力切碎，分成几份放。加入黄油，手动打蛋器搅打至巧克力全部融化，再微波炉热至自己满意可以喝的温度，香草，最后热完可以加几个棉花糖。
